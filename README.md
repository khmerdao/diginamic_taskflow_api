## Description

Dépôt de démarrage TypeScript pour le framework [Nest](https://github.com/nestjs/nest).

## Installation du projet

```bash
$ npm install
```

## Compiler et exécuter le projet

```bash
# développement
$ npm run start

# mode watch
$ npm run start:dev

# production
$ npm run start:prod
```

## Déploiement

Lorsque vous êtes prêt à déployer votre application NestJS en production, quelques étapes clés permettent de garantir une exécution aussi efficace que possible. Consultez la [documentation de déploiement](https://docs.nestjs.com/deployment) pour plus d'informations.

Si vous recherchez une plateforme cloud pour déployer votre application NestJS, découvrez [Mau](https://mau.nestjs.com), notre plateforme officielle pour déployer des applications NestJS sur AWS. Mau rend le déploiement simple et rapide, en quelques étapes seulement :

```bash
$ npm install -g @nestjs/mau
$ mau deploy
```

Avec Mau, vous pouvez déployer votre application en quelques clics, afin de vous concentrer sur le développement de fonctionnalités plutôt que sur la gestion de l'infrastructure.
